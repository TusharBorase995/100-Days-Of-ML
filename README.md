# 🤖 100 Days of Machine Learning (CampusX Challenge)

Welcome to my dedicated learning repository tracking my journey through the **100 Days of Machine Learning** program by Nitish Singh (CampusX). 

This repository serves as a transparent, day-by-day record of my dedication, code experiments, and conceptual mastery as I transition into the field of Applied Machine Learning and Data Science.

---

## 🎯 Project Objectives

Instead of just watching tutorials, I am actively building a production-ready foundation by focusing on three main pillars:
1. **The Mathematical "Why":** Deep diving into the geometry, calculus, and linear algebra behind algorithms rather than treating models like a black box.
2. **The Engineering "How":** Implementing robust, leak-proof Scikit-Learn pipelines, extensive feature engineering, and modular data processing tracking.
3. **Interview Readiness:** Summarizing every algorithm’s assumptions, advantages, limitations, and key hyperparameter tuning strategies for real-world deployments.

---

## 🗂️ Core Repository Architecture

Every daily milestone is strictly structured to ensure modularity and clean documentation:
* **`Day-XX_[Topic]/`** - Dedicated directory for each sequential day.
* **`Day-XX_Notes.md`** - Highly scannable technical summaries highlighting geometric intuition, core loss functions ($\text{Log Loss}$, $\text{MSE}$), and pros/cons.
* **`Practice_Notebook.ipynb`** - Extensively commented Jupyter notebooks exploring data cleaning, model benchmarking, and edge-case handling.
